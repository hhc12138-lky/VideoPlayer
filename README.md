# VideoPlayer
基于VLC的多进程视频播放服务端系统 | EPOLL、VLC、MySQL、HTTP、线程池、多进程
项目背景：本项目为在校期间的后端开发学习项目，基于C++、vlc和epoll实现一个播放系统服务端，本人独立完成整个服务端的架构设计与实现。 
主要工作： 
1.系统架构与进程设计：设计多进程架构，分离主进程、业务处理进程与日志打印进程，通过本地套接字进行进程间通信，提升系统稳定性与可维护性。
2.网络模块设计：基于原生socket类与epoll封装网络模块，实现reactor模型，监听客户端连接，并将客户端套接字发送给子进程处理； 
3.并发处理模块：封装epoll事件处理机制，提供增删改查接口，结合线程类封装线程池模块，使用STL封装线程库与线程池，通过锁机制保证线程安全，提供任务投递接口，实现任务分发线程池；
4.权限与安全机制：实现自动登录与权限校验模块，采用 OpenSSL 加密用户信息，并通过独立线程进行登录状态检查，防止非授权访问。
5.数据与协议处理：二次封装 HTTP 解析与 JSON 库，完成请求头、URL、状态码及消息体的解析；封装数据库操作类，支持连接管理、SQL 执行与事务处理。
